# Gather Market

Seller-owned physical product using Stripe Connect Express, direct charges, a snapshotted application fee, and seller-scoped fulfillment. Enable user selling, complete onboarding, moderate the product, then publish the owned offer.

The scenario previews Koru Clay's set at NZD 128.00 before Stripe shipping. Checkout adds the selected shipping rate and snapshots the platform application fee without increasing the buyer total. The offer-only checkout contract keeps every order structurally scoped to one seller.
