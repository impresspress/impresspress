# OrbitOps

Licensed SaaS subscription with monthly/annual offers, bounded seat input, optional one-time onboarding, a trial, promotion codes, and hosted Checkout. Publish new offer versions instead of mutating active customer pricing.

The deterministic journey chooses the annual offer, eight seats, and onboarding for USD 2720.00/year. Subscription access follows ordered Stripe subscription/invoice state and customers manage billing through the portal.
