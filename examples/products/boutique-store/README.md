# Marrow & Loom

Physical one-time product using select and boolean variables, conditional wool/monogram rows, automatic tax, and a tracked shipping option. Seed the fixture, publish `archive-jacket-offer`, and configure NZ/AU as allowed destinations.

Operations should fulfill from the immutable size, cloth, monogram, and shipping snapshot. The browser scenario previews an M/Wool monogram jacket at NZD 519.00; Stripe adds the selected shipping option and tax in hosted Checkout.
